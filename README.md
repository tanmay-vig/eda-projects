# ML PROJECTS

<div>Steps to run:</div>
<ol>
    <li>git clone the repo</li>
    <li>Set up the virtual env</li>
    <ul>

    <li>source myenv/bin/activate</li>
    </ul>
    <li>pip install -r requirements.txt</li>
    <li>Now you can run the files</li>
</ol>
